Sonos Roam SL 官方文檔

產品介紹
Sonos Roam SL 是一款輕量級便攜式智能揚聲器，設計用於在家中或外出時使用。

技術規格
- 頻率範圍：50Hz - 20kHz
- 藍牙版本：5.0
- WiFi：802.11ac (2x2 MIMO)
- 電池續航：最長 10 小時
- 重量：370 克
- 尺寸：120.9 x 78.5 x 62.8 毫米

連接方式
Roam SL 可以通過以下方式連接：
1. WiFi 網絡
2. 藍牙
3. AirPlay 2

音質特性
- 強大的低音和清晰的高音
- 專業級音質
- 防塵防水：IP67 等級

故障排除
如果 Roam SL 無法連接到 WiFi：
1. 確保揚聲器已打開
2. 檢查 WiFi 密碼是否正確
3. 重置揚聲器並重新配對
