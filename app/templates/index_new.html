Manual Packet Capture

Live capture disabled for security. Use the following command to capture packets:

sudo tcpdump -i en0 -s 65535 -G 5 -W 1 -w ~/Downloads/capture.pcap 'tcp port 1400 or (udp and port 1900)'

💡 macOS Tip: Use interface en0 (Wi‑Fi) or en4 (Ethernet). -G 5 means auto-stop after 5 seconds

Steps:
1️⃣ Run the command in your terminal
2️⃣ Wait 5 seconds (or press Ctrl+C to stop early)
3️⃣ Upload ~/Downloads/capture.pcap below for analysis

Sonos Web App

使用 Companion 模式整合 Sonos 官方 Web App，避免 iframe 被阻擋。

在新分頁開啟

Sonos 官方站點通常會阻擋 iframe 內嵌（X-Frame-Options / CSP），建議使用新視窗或新分頁模式。

快速操作

建議工作流程

  1. 先按「同分頁前往」進入 Sonos Web App。
  2. 需要回來時，在瀏覽器開啟你儲存的 `/return` 捷徑即可一鍵返回。
  3. 回到本地介面做診斷、群組、排程與自動化。
